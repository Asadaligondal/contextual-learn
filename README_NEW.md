# Contextual Learn

Contextual Learn is a Vite + React application for AI-powered tutoring and grading.

## Local development

1. Install dependencies

```bash
npm install
```

2. Start the dev server

```bash
npm run dev
```

3. Open the app at `http://localhost:8080`

## Notes

- This repository contains a demo auth mode that allows public sign-in for previews. For production, re-enable Firebase Auth or another secure auth provider.
- Secrets (such as `.env.local` and service account JSON) are ignored by `.gitignore`. Do not commit secrets.
